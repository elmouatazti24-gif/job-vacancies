# PhoneBookApp

A simple phone book application built with Python that allows users to store, search, update, and delete contact information. This project demonstrates basic CRUD operations, file handling, and object-oriented programming concepts.

## Features

* Add new contacts
* Search contacts by name
* Update existing contact details
* Delete contacts
* View all saved contacts
* Persistent storage using JSON files

## Technologies Used

* Python 3
* JSON
* Object-Oriented Programming

## Project Structure

```
PhoneBookApp/
├── main.py
├── contact_manager.py
├── contacts.json
├── requirements.txt
└── README.md
```

## Installation

1. Clone the repository:

```bash
git clone https://github.com/username/PhoneBookApp.git
```

2. Navigate to the project directory:

```bash
cd PhoneBookApp
```

3. Run the application:

```bash
python main.py
```

## Usage

1. Launch the application.
2. Select an option from the menu.
3. Enter contact information when prompted.
4. Save and manage contacts efficiently.
